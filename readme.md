# WeatherPredictor — 智能天气预测器

## 产品概述

WeatherPredictor 是一款基于 **ESP8266** 的智能桌面天气站，集成 OLED 显示、语音播报、室内温湿度传感和 MQTT 远程控制功能。设备自动从心知天气 API 获取实时天气和 3 日预报，通过 1.3 英寸 OLED 屏幕直观展示，并支持语音播报天气信息。

---

## 功能详述

### 1. 配网系统 (WiFiManager)

- 首次上电自动进入 **配网模式**，OLED 显示提示信息
- 手机/电脑连接热点 `AutoConnectAP`，自动弹出配置 portal
- 可配置参数：
  - **城市**（默认 `beijing`）
  - **MQTT 服务器地址**（默认 `192.168.5.4`）
- 配置保存后自动重启连接 Wi-Fi

### 2. 天气数据

- 数据源：[心知天气 API](https://www.seniverse.com/)（HTTPS + BearSSL）
- **当前天气**：温度、天气状况文字、天气代码
- **3 日预报**（今日/明日/后天）：最高温、最低温、白天天气、夜间天气、降水概率、风向、风力
- 自动更新间隔：
  - 当前天气：**10 分钟**
  - 未来天气：**6 小时**
- API 请求失败时自动重试（通过标志位触发）


### 3. 按键操作

| 按键 | 功能 |
|------|------|
| **电源开关** (GPIO16) | 切换开机/关机；开机时自动开启语音，关机时关闭屏幕和语音 |
| **翻页** (GPIO14) | 循环切换 4 个页面（0→1→2→3→0）；重置自动翻页计时 |
| **静音** (GPIO12) | 切换语音播报开启/关闭 |
| **播报** (GPIO15) | 触发当前页面的语音播报一次 |


### 4. 自动翻页

- 在**本地模式**下，页面每 **20 秒** 自动切换一次
- 手动翻页后自动翻页计时器**重置**
- 关机状态下自动翻页暂停
- **远控模式**下自动翻页暂停

### 5. MQTT 远程控制

#### 通信主题结构

详见MQTT消息结构

#### 远程控制命令

**控制命令** `clock/equip1/control`：
```json
{"command": [2, true, true, true, false]}
```
| 索引 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 0 | page | int | 跳转到指定页面 (0-3) |
| 1 | update_today | bool | 强制刷新当前天气 |
| 2 | update_future | bool | 强制刷新未来天气 |
| 3 | voice_trigger | bool | 触发语音播报 |
| 4 | voice_state | bool | 设置语音开关 |

**设置命令** `clock/equip1/set`：
```json
{"command": ["shanghai", true]}
```
| 索引 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 0 | city | string | 修改城市（同时更新 API URL 并刷新数据） |
| 1 | remote_mode | bool | true=远程模式，false=本地模式 |

> **模式说明**：
> - **本地模式**（默认）：按键可用，自动翻页开启
> - **远程模式**：禁用所有本地按键操作，仅受 MQTT 控制


### 6. 室内传感器

- DHT11 温湿度传感器，每 **10 秒** 读取一次
- 数据同时显示在 OLED 首页和 MQTT `state/sensor` 主题
- 读取失败时串口输出错误提示

### 7. 时间同步

- 通过 NTP 协议自动同步时间
- NTP 服务器：`ntp.aliyun.com`、`cn.ntp.org.cn`
- 时区：UTC+8（北京时间）
- 启动时**等待同步成功**后才继续加载天气数据

## 接线图

| ESP8266 引脚 | 连接设备 |
|:------------:|----------|
| GPIO13 (D7)  | DHT11 DATA |
| GPIO14 (D5)  | 翻页按键 → GND |
| GPIO15 (D8)  | 播报按键 → GND |
| GPIO12 (D6)  | 静音按键 → GND |
| GPIO16 (D0)  | 电源按键 → GND |
| GPIO1 (TX)   | 语音模块 RX |
| SDA (D2)     | OLED SDA |
| SCL (D1)     | OLED SCL |

> 按键均为高电平触发，按下时引脚接 VCC，松开时内部/外部下拉至 GND。
